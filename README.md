## Clone repository to home dir
```sh
git clone https://github.com/FruKun/DE_cfg.git ~/DE_cfg
```
## change packages
```sh
nano ~/DE_cfg/package-list
nano ~/DE_cfg/package-list-aur
```
## run install script
```sh
~/DE_cfg/install.sh
~/DE_cfg/install-packages.sh
```
## delete cfg
```sh
~/DE_cfg/uninstall-cfg.sh
```
